きもちチェックについて

「きもちチェック」は、子どもたちが自分のきもちをことばに見つけて、気持ちが楽になるヒントを試してみるきっかけを作るためのサービスです。学童保育や幼稚園・保育園、療育の現場で気軽に使えることを目指して制作に取り組みました。
本サービスは完全非営利であり、課金の必要や広告の表示は一切ありません。

※本サービスは開発途中です。予告なく機能が変更されたりサービスが終了したり、する可能性があります。

使い方

5つのしつもんに答えると、今のきもちにあった「とっこうやく（特効薬）」が10個提示されます。難しい操作はありません。

1
「はじめる」をおす
トップ画面のボタンをタップしてスタートします。
2
5つのしつもんにこたえる
きもち・からだのげんき・だれかといたいか・からだのかんじ・いましたいこと、の順に選ぶだけ。
3
とくこうやくをみる
今のきもちの分析と、それにぴったりな10個のヒントが画面に表示されます。気になるものを子どもと一緒にみつけて、やってみましょう。
4
もういちどやる
ボタンを押すとスタート画面に戻ります。同じ回答をしても、毎回違う特効薬を見ることができます。

私たちのゼミについて

本サイトは横浜保育教育専門学校 にじいろゼミが制作しました。
にじいろゼミでは日々、発達多様性や療育についての研究活動を行っています。

内部の仕組み

このアプリはHTMLとJavaScriptだけで動いており、サーバーへの通信は一切行いません。回答データは外部に送信されず、ブラウザ上だけで処理されます。

"特効薬"の選ばれかた

約100個の特効薬には、それぞれ5つの質問に対応した「タグ」が設定されています。5つの回答と各特効薬のタグを照合してスコアを計算し、スコアが高いものほど選ばれやすい「重み付きランダム選択」で10個を決定します。

しつもんタグの例
いまのきもちhappy / sad / angry / anxious / any
からだのげんきhigh / mid / low / verylow / any
だれかといたいかsocial / alone / neutral / any
からだのかんじheart / breath / stomach / fine / any
いましたいことsleep / play / talk / nothing / any
